# 🔍 SimplyScan

A lightweight multithreaded port scanner written in Python.

## 🚀 Quick Start

```bash
python3 simplyscan.py
```


---
## ⚙️ Features


* Scans ports 1–1024
* Identifies common services
* Multi-threaded scanning
* Simple CLI interface
